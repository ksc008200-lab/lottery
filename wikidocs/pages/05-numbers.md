## Two Number Systems! · 두 가지 숫자 체계!

Korean has **two completely separate number systems**, each used in different situations. You must learn both!

한국어에는 **완전히 다른 두 가지 숫자 체계**가 있으며, 각각 다른 상황에서 사용됩니다. 둘 다 배워야 합니다!

| System | 한국어 | Used For |
| :---: | :---: | :---: |
| Sino-Korean 한자어 | 일 이 삼 사... | Dates, money, phone numbers, minutes, floors, addresses |
| Native Korean 고유어 | 하나 둘 셋 넷... | Counting objects, hours (time), age (casual), people |

### Building Larger Numbers · 큰 숫자 만들기

Korean combines units differently from English. **10,000 (만) is a key unit** — not 1,000.

    11 = 십일 (sip-il) — ten-one
    25 = 이십오 (i-sip-o) — two-ten-five
    100 = 백 (baek)
    350 = 삼백오십 (sam-baek-o-sip)
    1,000 = 천 (cheon)
    2,500 = 이천오백 (i-cheon-o-baek)
    10,000 = 만 (man)
    50,000 = 오만 (o-man)
    100,000 = 십만 (sip-man)
    1,000,000 = 백만 (baek-man)

## Counting Units · 세는 단위 (수량사)

Korean uses **counters** after numbers when counting objects. The number form changes slightly with counters.

한국어는 물건을 셀 때 숫자 뒤에 **단위명사**를 붙입니다. 숫자 형태가 약간 바뀝니다.

### Native Korean Number Shortcuts Before Counters

| Number | Standalone | Before Counter |
| :---: | :---: | :---: |
| 1 | 하나 | 한 |
| 2 | 둘 | 두 |
| 3 | 셋 | 세 |
| 4 | 넷 | 네 |
| 20 | 스물 | 스무 |

## Telling Time · 시간 말하기

**Hours → Native Korean** | **Minutes → Sino-Korean**

시(hour) → 고유어 | 분(minute) → 한자어

| Time | 한글 | Pronunciation |
| :---: | :---: | :---: |
| 1:00 | 한 시 | han si |
| 2:00 | 두 시 | du si |
| 3:00 | 세 시 | se si |
| 6:00 | 여섯 시 | yeo-seot si |
| 12:00 | 열두 시 | yeol-du si |
| 1:30 | 한 시 삼십 분 | han si sam-sip bun |
| 3:15 | 세 시 십오 분 | se si si-bo bun |
| 6:45 | 여섯 시 사십오 분 | yeo-seot si sa-si-bo bun |

    💡 Half past = 반 (ban)
    두 시 반 = 2:30 (du si ban)

    AM = 오전 (o-jeon)    PM = 오후 (o-hu)
    오전 열 시 = 10:00 AM
    오후 세 시 = 3:00 PM

## Dates · 날짜

Dates use Sino-Korean numbers.

날짜는 한자어 숫자를 씁니다.

    Year (년): 이천이십육 년 = 2026년
    Month (월): 사 월 = April (4월)
    Day (일): 십일 일 = 11th (11일)

| Month | 한글 | Pronunciation | Month | 한글 | Pronunciation |
| :---: | :---: | :---: | :---: | :---: | :---: |
| January | 일월 | il-wol | July | 칠월 | chil-wol |
| February | 이월 | i-wol | August | 팔월 | pal-wol |
| March | 삼월 | sam-wol | September | 구월 | gu-wol |
| April | 사월 | sa-wol | October | 시월 | si-wol |
| May | 오월 | o-wol | November | 십일월 | si-bil-wol |
| June | 유월 | yu-wol | December | 십이월 | si-bi-wol |

> **Note:** June = 유월 (NOT 육월), October = 시월 (NOT 십월) — irregular contractions!

## Quick Reference · 빠른 참고

    When to use Sino-Korean (한자어):
    ✅ Money: 오만 원 (₩50,000)
    ✅ Dates: 사월 십일 일 (April 11th)
    ✅ Minutes: 삼십 분 (30 minutes)
    ✅ Phone numbers: 공일공-일이삼사-오육칠팔
    ✅ Addresses/floors: 삼 층 (3rd floor)
    ✅ Math: 삼 더하기 사 = 칠 (3+4=7)

    When to use Native Korean (고유어):
    ✅ Counting objects: 사과 세 개 (3 apples)
    ✅ Hours: 두 시 (2 o'clock)
    ✅ Age (casual): 스물다섯 살 (25 years old)
    ✅ People (informal): 두 명 (2 people)
