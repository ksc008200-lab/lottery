# Ch29: Spelling Rules — 맞춤법

<div class="tip">
      💡 <strong>Why Spelling Matters in Korean!</strong><br>
      Even native Koreans struggle with 맞춤법! A single wrong letter completely changes the meaning — <span class="hangul-cell">됐어</span> vs <span class="hangul-cell">됬어</span>, <span class="hangul-cell">안 해</span> vs <span class="hangul-cell">않해</span>. Koreans judge each other's writing by spelling — get it right and sound educated! 🎓
    </div>

    <!-- 되 vs 돼 -->
    <h3 style="margin-top:32px;">🔥 #1 Most Common Mistake — 되 vs 돼</h3>
    <p>This is the #1 spelling mistake even Koreans make! Here's the easy rule:</p>

    <div style="display:grid; grid-template-columns:1fr 1fr; gap:16px; margin:20px 0;">
      <div class="card" style="padding:20px; border-left:4px solid #1A4A8A;">
        <div style="font-size:14px; font-weight:700; color:#1A4A8A; margin-bottom:10px;">되 · doe</div>
        <div class="hangul-cell" style="font-size:32px;">되</div>
        <div style="font-size:12px; color:#666; margin-top:8px;">= "하" 로 바꿔도 자연스러움</div>
        <div style="margin-top:10px; font-size:13px;">
          <div>✅ 안 <strong>되</strong>요 (안 하요 → ✅)</div>
          <div>✅ 이렇게 하면 <strong>되</strong>나요?</div>
          <div>✅ <strong>되</strong>게 맛있어</div>
        </div>
      </div>
      <div class="card" style="padding:20px; border-left:4px solid #C0392B;">
        <div style="font-size:14px; font-weight:700; color:#C0392B; margin-bottom:10px;">돼 · dwae</div>
        <div class="hangul-cell" style="font-size:32px;">돼</div>
        <div style="font-size:12px; color:#666; margin-top:8px;">= 되 + 어 = "해" 로 바꿔도 자연스러움</div>
        <div style="margin-top:10px; font-size:13px;">
          <div>✅ 안 <strong>돼</strong>요 (안 해요 → ✅)</div>
          <div>✅ 잘 <strong>됐</strong>어! (됐 = 되었)</div>
          <div>✅ 어떻게 <strong>돼</strong>?</div>
        </div>
      </div>
    </div>

    <div style="background:#fffbea; border-radius:12px; padding:16px; margin:12px 0; border-left:4px solid #E8C84A;">
      <div style="font-weight:700; margin-bottom:8px;">💡 쉬운 방법 — The "해" Test!</div>
      <p style="margin:0; font-size:14px;">문장에서 <strong>되/돼</strong> 자리에 <strong>"해"</strong>를 넣어보세요:</p>
      <p style="margin:8px 0 0; font-size:14px;">• "안 <em>해</em>요" → 자연스러움 → <strong>돼</strong> ✅</p>
      <p style="margin:4px 0 0; font-size:14px;">• "안 <em>하</em>요" → 자연스러움 → <strong>되</strong> ✅</p>
    </div>

    <!-- 안 vs 않 -->
    <h3 style="margin-top:32px;">🔥 #2 — 안 vs 않</h3>

    <div style="display:grid; grid-template-columns:1fr 1fr; gap:16px; margin:20px 0;">
      <div class="card" style="padding:20px; border-left:4px solid #1A4A8A;">
        <div style="font-size:14px; font-weight:700; color:#1A4A8A; margin-bottom:8px;">안 · an</div>
        <div class="hangul-cell" style="font-size:28px;">안</div>
        <div style="font-size:12px; color:#666; margin-top:6px;">부사 — 동사/형용사 <strong>앞</strong>에</div>
        <div style="margin-top:10px; font-size:13px; line-height:1.8;">
          ✅ <span class="hangul-cell">안 먹어</span> (안 + 먹어)<br>
          ✅ <span class="hangul-cell">안 가요</span> (안 + 가요)<br>
          ✅ <span class="hangul-cell">안 예뻐</span>
        </div>
      </div>
      <div class="card" style="padding:20px; border-left:4px solid #C0392B;">
        <div style="font-size:14px; font-weight:700; color:#C0392B; margin-bottom:8px;">않 · an-h</div>
        <div class="hangul-cell" style="font-size:28px;">않</div>
        <div style="font-size:12px; color:#666; margin-top:6px;">보조용언 — <strong>-지 않다</strong> 형태로</div>
        <div style="margin-top:10px; font-size:13px; line-height:1.8;">
          ✅ <span class="hangul-cell">먹지 않아</span> (-지 않다)<br>
          ✅ <span class="hangul-cell">가지 않아요</span><br>
          ✅ <span class="hangul-cell">예쁘지 않아</span>
        </div>
      </div>
    </div>

    <div style="background:#f9f9f9; border-radius:12px; padding:14px; margin:12px 0; border:1px solid #e0ddd5;">
      <div style="font-weight:700; color:#C0392B; margin-bottom:6px;">❌ 흔한 실수 Common Mistakes</div>
      <div style="font-size:13px; line-height:2;">
        ❌ <span class="hangul-cell">않 먹어</span> → ✅ <span class="hangul-cell">안 먹어</span><br>
        ❌ <span class="hangul-cell">먹지 안아</span> → ✅ <span class="hangul-cell">먹지 않아</span><br>
        ❌ <span class="hangul-cell">않예뻐</span> → ✅ <span class="hangul-cell">안 예뻐</span>
      </div>
    </div>

    <!-- 이/히 -->
    <h3 style="margin-top:32px;">🔥 #3 — -이 vs -히 (부사 만들기)</h3>
    <p>형용사에 붙여 부사를 만들 때 — 이/히 중 어떤 걸 쓸까요?</p>

    <table style="width:100%; border-collapse:collapse; margin:16px 0; font-size:14px;">
      <thead>
        <tr style="background:#145A32; color:white;">
          <th style="padding:10px; text-align:left;">-이 사용</th>
          <th style="padding:10px; text-align:left;">-히 사용</th>
        </tr>
      </thead>
      <tbody>
        <tr style="background:#f9f9f9;">
          <td style="padding:10px;">받침이 ㄱ,ㅂ → <span class="hangul-cell">깨끗이</span>, <span class="hangul-cell">가볍이</span>❌→<span class="hangul-cell">가벼이</span></td>
          <td style="padding:10px;">-하다 형용사 → <span class="hangul-cell">조용히</span>, <span class="hangul-cell">솔직히</span></td>
        </tr>
        <tr>
          <td style="padding:10px;"><span class="hangul-cell">많이</span> (many/much)</td>
          <td style="padding:10px;"><span class="hangul-cell">천천히</span> (slowly)</td>
        </tr>
        <tr style="background:#f9f9f9;">
          <td style="padding:10px;"><span class="hangul-cell">같이</span> (together)</td>
          <td style="padding:10px;"><span class="hangul-cell">특히</span> (especially)</td>
        </tr>
        <tr>
          <td style="padding:10px;"><span class="hangul-cell">일찍이</span> (early)</td>
          <td style="padding:10px;"><span class="hangul-cell">확실히</span> (certainly)</td>
        </tr>
      </tbody>
    </table>

    <!-- 헷갈리는 쌍 -->
    <h3 style="margin-top:32px;">😵 헷갈리는 맞춤법 쌍 · Confusing Pairs</h3>

    <table style="width:100%; border-collapse:collapse; margin:16px 0; font-size:14px;">
      <thead>
        <tr style="background:#1E8449; color:white;">
          <th style="padding:10px;">틀린 표현 ❌</th>
          <th style="padding:10px;">올바른 표현 ✅</th>
          <th style="padding:10px;">설명</th>
        </tr>
      </thead>
      <tbody>
        <tr style="background:#f9f9f9;">
          <td style="padding:10px; color:#C0392B;"><span class="hangul-cell">왠만하면</span></td>
          <td style="padding:10px; color:#1A6A3A;"><span class="hangul-cell">웬만하면</span></td>
          <td style="padding:10px;">웬 = "어떤" / 왠 = "왜인지"</td>
        </tr>
        <tr>
          <td style="padding:10px; color:#C0392B;"><span class="hangul-cell">왠지</span> ❌</td>
          <td style="padding:10px; color:#1A6A3A;"><span class="hangul-cell">왠지</span> ✅</td>
          <td style="padding:10px;">왠지 = "왜인지" (이건 맞음!)</td>
        </tr>
        <tr style="background:#f9f9f9;">
          <td style="padding:10px; color:#C0392B;"><span class="hangul-cell">어떻해</span></td>
          <td style="padding:10px; color:#1A6A3A;"><span class="hangul-cell">어떡해</span></td>
          <td style="padding:10px;">어떡해 = 어떻게 해 (줄임)</td>
        </tr>
        <tr>
          <td style="padding:10px; color:#C0392B;"><span class="hangul-cell">어떻게해</span></td>
          <td style="padding:10px; color:#1A6A3A;"><span class="hangul-cell">어떡해</span> / <span class="hangul-cell">어떻게 해</span></td>
          <td style="padding:10px;">붙이면 안 됨</td>
        </tr>
        <tr style="background:#f9f9f9;">
          <td style="padding:10px; color:#C0392B;"><span class="hangul-cell">맞히다</span> (정답 )</td>
          <td style="padding:10px; color:#1A6A3A;"><span class="hangul-cell">맞히다</span> ✅ / <span class="hangul-cell">맞추다</span></td>
          <td style="padding:10px;">맞히다 = 정답 / 맞추다 = 조합</td>
        </tr>
        <tr>
          <td style="padding:10px; color:#C0392B;"><span class="hangul-cell">며칠</span> vs <span class="hangul-cell">몇일</span></td>
          <td style="padding:10px; color:#1A6A3A;"><span class="hangul-cell">며칠</span> ✅</td>
          <td style="padding:10px;">"몇일"은 틀린 표현</td>
        </tr>
        <tr style="background:#f9f9f9;">
          <td style="padding:10px; color:#C0392B;"><span class="hangul-cell">설레임</span></td>
          <td style="padding:10px; color:#1A6A3A;"><span class="hangul-cell">설렘</span> ✅</td>
          <td style="padding:10px;">설레다 → 설렘 (아이스크림 이름은 예외)</td>
        </tr>
        <tr>
          <td style="padding:10px; color:#C0392B;"><span class="hangul-cell">금새</span></td>
          <td style="padding:10px; color:#1A6A3A;"><span class="hangul-cell">금세</span> ✅</td>
          <td style="padding:10px;">금세 = 금방, 곧</td>
        </tr>
        <tr style="background:#f9f9f9;">
          <td style="padding:10px; color:#C0392B;"><span class="hangul-cell">어의없다</span></td>
          <td style="padding:10px; color:#1A6A3A;"><span class="hangul-cell">어이없다</span> ✅</td>
          <td style="padding:10px;">어이 = 기가 막힘 (어의=왕의 의사)</td>
        </tr>
        <tr>
          <td style="padding:10px; color:#C0392B;"><span class="hangul-cell">오랫만에</span></td>
          <td style="padding:10px; color:#1A6A3A;"><span class="hangul-cell">오랜만에</span> ✅</td>
          <td style="padding:10px;">오랜만 = 오랜 시간 만에</td>
        </tr>
      </tbody>
    </table>

    <!-- 띄어쓰기 -->
    <h3 style="margin-top:32px;">📏 띄어쓰기 · Spacing Rules</h3>
    <p>Korean spacing is one of the hardest parts even for natives. Key rules:</p>

    <div style="display:grid; grid-template-columns:repeat(auto-fit,minmax(280px,1fr)); gap:14px; margin:20px 0;">
      <div class="card" style="padding:16px;">
        <div style="font-weight:700; color:#1A4A8A; margin-bottom:8px;">✅ 띄어 써야 해요 (Space needed)</div>
        <div style="font-size:13px; line-height:2;">
          <span class="hangul-cell">밥 먹었어</span> — 동사는 띄어<br>
          <span class="hangul-cell">나는 학생이 아니야</span><br>
          <span class="hangul-cell">가장 좋아하는 것</span><br>
          <span class="hangul-cell">할 수 있어</span> — 수, 것 띄어
        </div>
      </div>
      <div class="card" style="padding:16px;">
        <div style="font-weight:700; color:#C0392B; margin-bottom:8px;">✅ 붙여 써야 해요 (No space)</div>
        <div style="font-size:13px; line-height:2;">
          <span class="hangul-cell">이것</span>, <span class="hangul-cell">그것</span>, <span class="hangul-cell">저것</span><br>
          <span class="hangul-cell">먹었는데</span> — 어미는 붙여<br>
          <span class="hangul-cell">예쁘지만</span> — 접속어미 붙여<br>
          <span class="hangul-cell">학교에서</span> — 조사 붙여
        </div>
      </div>
    </div>

    <!-- 낫다 낮다 났다 -->
