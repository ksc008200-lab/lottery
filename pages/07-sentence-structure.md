# Ch07: Sentence Structure — 문장 구조

<div class="card red">
      <div class="card-title">🤯 The Biggest Difference from English</div>
      <p>English: Subject → <strong>Verb</strong> → Object &nbsp;&nbsp; (I <strong>eat</strong> pizza)</p>
      <p>Korean: Subject → Object → <strong>Verb LAST!</strong> &nbsp;&nbsp; (나는 피자를 <strong>먹어요</strong>)</p>
      <p class="kr">영어는 동사가 가운데, 한국어는 동사가 항상 마지막! 끝까지 들어야 무슨 행동인지 알 수 있어요.</p>
    </div>
